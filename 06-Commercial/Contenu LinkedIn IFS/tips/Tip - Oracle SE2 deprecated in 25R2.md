**Pilier** : Upgrades & SU Management

💡 **IFS Cloud tip**

A licensing landmine hiding in IFS Cloud upgrades: Oracle **Standard Edition 2** support is being deprecated, with **Enterprise Edition** becoming mandatory. `[à vérifier : 25R2 / sept. 2025]`

• Check which DB edition each environment runs **now**
• EE changes the licensing-cost conversation — loop in procurement early
• Don't discover this on the upgrade weekend

The technical upgrade is rarely the real risk. It's the dependency nobody mapped — DB edition, integrations, custom logic — that turns a routine SU into a fire drill.
