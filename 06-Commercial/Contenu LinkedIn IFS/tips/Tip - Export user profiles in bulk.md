💡 **IFS Cloud tip**

There's no one-click export for multiple user profiles in IFS Cloud — but you don't actually need one.

• Export the profile data with a **Quick Report** over the client profile nodes
• Reshape it in Excel and re-import as **XML**
• You can push specific sections — bookmarks, table-view layouts — into many users at once

Perfect for onboarding a team on a consistent layout, or carrying personalizations from TEST to PROD instead of rebuilding them by hand. When a UI action is missing, the underlying data is usually still reachable.
