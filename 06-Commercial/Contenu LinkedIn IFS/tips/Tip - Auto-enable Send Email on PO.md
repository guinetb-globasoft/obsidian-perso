**Pilier** : Custom Fields & Workflow

💡 **IFS Cloud tip**

A recent release made users manually flip the "Send Email" toggle every time they email a PO — and the report rules can't pre-set it. Death by a thousand clicks, and an easy step to forget.

When standard config can't default a value, a lightweight **business process automation (workflow)** usually can:

• Set the toggle to true whenever the print/email dialog runs
• No core customization
• Consistent behaviour for every user, no training note required

The broader principle: if config won't hold a default, reach for a workflow before you reach for a mod.
