💡 **IFS Cloud tip**

Lobby page parameters showing in an awkward order and you can't drag them around? That's expected behaviour, not a missing skill.

• Page parameters render in the **order they were defined / added**
• Page Designer rearranges layouts and visual elements — but **not** the parameter order
• Reordering later often means recreating the parameters

So decide the parameter sequence **up front**, in the order you want users to see it. It's one of the few things in a lobby you can't fix with a quick drag-and-drop afterwards — design intent beats rework.
