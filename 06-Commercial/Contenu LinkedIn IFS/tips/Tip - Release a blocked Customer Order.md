💡 **IFS Cloud tip**

A Customer Order stuck in **Blocked** status? No admin gymnastics needed — use the **Blocked Info** button on the CO header.

• It opens **Handle Blocked Customer Orders**
• Select the order line → **Release from Credit Check** → the order flows again

The release path is built into the order itself. Knowing this one button beats raising a ticket every month-end when credit checks pile up — and it puts control back with the order desk.
