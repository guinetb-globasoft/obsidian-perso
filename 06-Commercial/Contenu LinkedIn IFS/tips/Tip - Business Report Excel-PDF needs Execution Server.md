💡 **IFS Cloud tip**

Want end users to view or forward a Business Report as a finished **Excel or PDF**? That capability needs the **BR Execution Server** installed and configured.

• The report is forwarded to the execution server for full rendering
• The generated file lands in the **report archive**, retrievable as Excel or PDF

No execution server, no full render. If "export to Excel/PDF" looks broken or comes back empty, it's usually an **infrastructure gap**, not a user-permission problem. Check the server before the role.
