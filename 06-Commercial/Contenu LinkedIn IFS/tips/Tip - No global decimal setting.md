💡 **IFS Cloud tip**

Looking for one setting to standardize decimal precision across *all* fields in IFS Cloud? It doesn't exist — and a day can disappear hunting for it.

• The "Currency Formatting Option" affects **currency** fields only
• Quantities and other numerics aren't covered
• Precision is defined **per attribute / domain** in the data model

So set it where the data is modeled, not in a global switch you'll never find. Telling apart "this is config" from "this is data-model" is half of working fast in IFS.
