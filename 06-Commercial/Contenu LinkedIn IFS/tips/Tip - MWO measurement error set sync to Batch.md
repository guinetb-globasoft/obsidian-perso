💡 **IFS Cloud tip**

Errors when entering a new measurement in **MWO** (Maintenance Work Order app)? Look at your **sync rules for Equipment objects**.

• Set the Equipment-object sync rules to **Batch** (for the app in use, e.g. MaintEngApp) and it resolves

Mobile issues in IFS often trace back to **sync-rule mode**, not the transaction. When a mobile action errors but the desktop equivalent works fine, check the sync configuration before anything else.
