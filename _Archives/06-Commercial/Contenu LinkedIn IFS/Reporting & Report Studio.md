# Pilier — Reporting & Report Studio

**Pourquoi ça marche** : sujet brûlant avec la fin de Crystal Reports (25r1) + migration vers Report Studio / Quicksight. Beaucoup de questions = beaucoup d'anxiété = bon reach.

## Brouillon de post (EN)
> **Crystal Reports is being removed in IFS Cloud 25R1. If you haven't planned the move, now's the time.**
>
> What teams are hitting in the transition:
> • Quick Reports that silently stop emailing on schedule
> • Report Studio image functions (Media / DBImage) not rendering
> • Currency code formatting that won't behave
> • "Where are my REP reports even stored now?" (25R2 changed it)
>
> A migration checklist:
> 1️⃣ Inventory every Crystal report still in use `[à préciser]`
> 2️⃣ Re-platform to Report Studio / Quicksight by priority
> 3️⃣ Re-test scheduling + email delivery (the #1 thing that breaks)
>
> How far along is your Crystal → Report Studio migration? 👇
