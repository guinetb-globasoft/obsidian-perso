# Contenu LinkedIn — Tips IFS (source : community.ifs.com)

> Idées de posts LinkedIn **en anglais**, **tips IFS purs**, sourcées depuis les
> questions récurrentes du forum IFS Community. Chaque note de pilier contient un
> brouillon de post + les **fils sources** (à vérifier/approfondir avant publication).

## Méthode
- Source = sujets réels du forum (250 sujets scrapés sur 5 catégories le 2026-05-30).
- Principe : une question posée par des dizaines d'utilisateurs = un sujet que ton audience cherche → bon post.
- Format type : **Hook** (le problème) → **2-4 points concrets** → **mini-CTA / question**.
- Ton : expert, utile, non promotionnel. Le produit (ERP Control) se vend tout seul sur les piliers droits/imports.

## Piliers identifiés
1. [[Reporting & Report Studio]] — Crystal sunset, Quick Report emails, formats
2. [[Permissions & Roles]] — permission sets, functional roles, row level security *(angle produit)*
3. [[Data Import & APIs]] — Excel add-in, REST/OAuth, import vouchers *(angle produit)*
4. [[Upgrades & SU Management]] — dépendances, ordre de déploiement, sizing
5. [[E-Invoicing & Compliance]] — FR/DE/PL, ISO 20022, PEPPOL
6. [[Aurena & Personalization]] — sort order, LOV filters, conditional formatting
7. [[Custom Fields & Workflow]] — defaults, mandatory, events → workflow
